C++ 基礎 ① : 開發環境

整合開發環境(IDE):

IDE 將程式碼編輯、編譯、調試等功能整合在一起，能提升開發效率。以下是常見的 IDE：



Visual Studio Code

  • 擁有智慧程式碼提示、調試工具,支援多種程式語言
  • 內建命令列工具和 Git 版本控制系統。


CLion

  • 由 JetBrains 開發，具備程式碼補全、重構功能。
  • 提供對 Cmake 的支持


Code::Blocks

  • 喀，不知道。(應該也很厲害)

大家應該都會使用 Visual Studio Code 吧！反正我最喜歡用Clion(其實我所有編程開發都是依靠JetBrains的)。



接下來示範一下Clion的安裝(因為應該沒有人的電腦是沒有VSCode的吧)：


第一步：來到 JetBrains 官方網站中的Clion下載區 jetbrains.com/clion/。

第二步：下載Clion安裝器，通常是.exe檔案，安裝部份自己處理吧！(其實這沒必要說)。

最後：寫一條自以為很厲害的代碼(有點扯)。 


#include <bits/stdc++>

using namespace std;

int main(){

    cout<<"Hello World!"<<endl; //輸出Hello World!

    return 0;

}
